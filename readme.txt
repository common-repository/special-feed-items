Prepent unicode characters before titles in your RSS feed.

== Description ==

Ever wanted to put special emphasis on some of your content in your RSS feed? Well this plugin allows you to put a special symbol, for instance a &hearts;, in front of the titles of especially categorized content in your feed.

== Installation ==

1. Upload the whole folder to your wp-content/plugins/ folder
2. Go to your Wordpress admin -> Plugins and activate the plugin.
3. That should be it.
